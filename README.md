# Java + Docker + Jenkins + Maven (CI/CD Project)

This is a real-time DevOps CI/CD project.

## Tools Used
- Java (Spring Boot)
- Maven
- Docker
- Jenkins
- GitHub

## CI/CD Flow
Developer → GitHub → Jenkins → Maven → Docker → Container

## Jenkins Stages
1. Clone code from GitHub
2. Build application using Maven
3. Create Docker image
4. Run Docker container

## Output
Open browser:
http://<jenkins-ip>:8080

You will see:
✅ Deployed using Docker + Jenkins + Maven CI/CD Pipeline!
